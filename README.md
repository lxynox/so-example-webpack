# so-example-webpack
Demo for a react-webpack configuration (using Webpack v2.6)

### Usage

Install everything, Run

`yarn`

If in  `Dev mode`, Run

`yarn dev`

If in `Prod mode`, Run

`yarn prod`

Wait and see ...
